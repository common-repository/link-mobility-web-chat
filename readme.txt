Link Mobility Web Chat Widget configuration settings

== Description ==
This plugin allows quick configuration of the Link Mobility Web Chat Widget by using the simple parameters you can obtain from your account publish dialog.
Copy and paste the required Link Mobility parameters and the Chatbot snippet will be automatically generated in your pages.

This plugin relies on the Link Mobility chatbot platform for every chatbot functionality. Please make sure to check Link Mobility main website at https://chatexperience.linkmobility.com.
